# 關掉Google表單連結預覽 (Google Sheets Link Preview Disabler)

這是一個 Chrome 擴充元件，專門用來**關閉 Google 試算表 (Google Sheets) 中滑鼠游標移到連結時自動彈出的預覽視窗**。

## 功能特色 (Features)

1.  **預設關閉預覽**：當滑鼠移到連結上時，不再自動彈出煩人的預覽視窗。
2.  **按住 Ctrl 暫時顯示**：如果你需要查看預覽，只需按住鍵盤上的 `Ctrl` 鍵，再將滑鼠移到連結上即可顯示。
3.  **快速開關**：點擊擴充元件圖示，可以隨時啟用或停用此功能。
4.  **支援深色模式圖示**：介面圖示針對深色/淺色主題優化。

## 安裝方式 (Installation)

由於此擴充元件尚未上架至 Chrome 線上應用程式商店，請依照以下步驟安裝：

1.  下載此專案的程式碼（或是 Clone 這個 Repo）。
2.  開啟 Chrome 瀏覽器，在網址列輸入 `chrome://extensions` 並按下 Enter。
3.  開啟右上角的 **「開發人員模式」 (Developer mode)**。
4.  點擊左上角的 **「載入未封裝項目」 (Load unpacked)**。
5.  選擇此專案的資料夾 (`Google excel delink`)。
6.  完成！

## 使用說明 (Usage)

- **一般使用**：安裝後即生效，滑鼠移過連結將不會有反應。
- **需要預覽時**：按住 `Ctrl` 鍵不放，滑鼠移過連結。
- **暫時停用外掛**：點擊瀏覽器右上角的拼圖圖示，找到本擴充元件，點開開關將其關閉即可。
